# Text Summarizer & Language Translator

A Python project that summarizes text and translates it into multiple languages using NLP techniques.

## Features

- Extractive text summarization using TF-IDF
- Language detection
- Text translation using Google Translate
- GUI interface using Tkinter

## Installation

1. Clone the repository
2. Create a virtual environment:
   ```bash
   python -m venv venv
   ```

## Activate venv:

venv\Scripts\activate

## Install dependencies:

pip install -r requirements.txt

## Usage

python Main.py

## Dependencies

**Python 3.11**
**numpy**
**nltk**
**scikit-learn**
**googletrans==4.0.0-rc1**
